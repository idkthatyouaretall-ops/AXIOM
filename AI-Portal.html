Setup Help

Each AI uses your Groq API key from its settings panel.

How to get a Groq API key

  1. Go to console.groq.com/keys and sign in or create a Groq account.
  2. Create a new API key and copy it.
  3. Open one of the AIs, go to Settings, paste the key into the Groq API key field, and save.
  4. If models do not load, refresh the AI page and check that the key was copied completely.

Which one should I use?

  • Use Llama for quick answers and lightweight chats.
  • Use Nexus when images, fallback reliability, or multi-agent reasoning matter.
  • Use Boundless when you want files, artifacts, previews, or editable generated code.
